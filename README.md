EcStock
================
Desktop application built with scala fx.

# Planning
[Planning Board](https://github.com/kapit4n/ec-stock/projects/1)

## Prerequisites
* sbt
* openjfx

## Install and Run
* sbt run

## Package
* sbt universal:packageBin

## Features
* Register products
* Register vendors
* Register customers
* Purchase Orders
* Inventory Check
* Orders/Shopping card
* Customer/orders chart report
* Product/Order Report
